# Goals Review

Use this module when the user asks how they are doing overall, wants progress on goals, or wants a broader review of health metrics, activity targets, rehab progress, or event preparation.

## Goal
Translate training and recovery data into a clear progress view that helps the user decide what to keep, change, or emphasize next.

## Goal categories
- health metrics
- activity volume goals
- event training goals
- rehab / mobility goals

## Workflow
1. Identify which goals are active from the profile
2. Pull the best available recent data
3. Compare recent actuals against stated targets
4. Summarize trend direction, not just snapshots
5. Identify the biggest blocker or strongest signal
6. Recommend the next adjustment if one is warranted

## Output shape
A good goals review usually includes:
- short top-line summary
- per-goal status
- trend direction
- key blocker or driver
- suggested next focus

## Rules
- be honest about uncertainty if data is incomplete
- avoid pretending precision that does not exist
- do not drown the user in numbers if the takeaway is simple
- if progress is mixed, say so plainly
- connect outcomes back to adherence and recovery patterns where relevant

## Tone
- strategic and clear
- less daily-coach energy, more scoreboard energy
- still human and grounded
