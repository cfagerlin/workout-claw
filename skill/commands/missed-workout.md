# Missed Workout

Use this module when the user skips a planned workout, misses part of a session, or asks how to adjust the week after falling behind.

## Goal
Respond without guilt, preserve week integrity, and decide what to reschedule versus what to let go.

## Core principles
- acknowledge the miss without drama
- protect the quality of the remaining week
- do not reflexively cram every missed session into later days
- key sessions deserve more rescue effort than accessory work
- rehab / Prehab consistency often matters more than ego-driven make-up sessions

## Workflow
1. Identify what was missed
2. Determine the priority of that session
3. Review the remaining calendar and recovery state
4. Decide whether to reschedule, modify, or drop it
5. Show the adjusted plan briefly and clearly

## Priority framework
### Usually worth rescheduling
- key programmed strength session
- key endurance session tied to an event goal
- rehab / Prehab session if consistency is slipping

### Usually okay to let go
- low-priority accessory work
- bonus conditioning
- redundant volume that would overload the rest of the week

## Adjustment rules
- never create a revenge-schedule because the user missed one day
- prefer preserving the next key session over forcing a make-up session
- if moving the missed session would materially harm recovery, drop or shrink it
- use minimum-dose alternatives when that preserves momentum better than a full reschedule

## Output style
Keep it concise:
- what was missed
- what will happen now
- what changed in the week

## Tone
- no guilt trips
- no fake positivity
- calm, practical, forward-looking
